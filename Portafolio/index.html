Swoopy

Swoopy es una aplicación tipo red social donde los usuarios pueden interactuar mediante publicaciones, comentarios, reacciones y mensajes. Integra un sistema interno llamado Orbix que impulsa la participación dentro de la plataforma, además de un marketplace enfocado en la venta de productos entre usuarios. Todo diseñado para ofrecer una experiencia rápida, intuitiva y fácil de usar.

Departamentos Velamar

Sistema web que centraliza la administración y visualización de departamentos. El usuario puede explorar cada propiedad, revisar disponibilidad, consultar precios, ver imágenes y obtener información clave. Los administradores tienen acceso a un panel para modificar datos, actualizar inventarios, gestionar calendarios y mantener todo el contenido en sincronía.

H. Ayuntamiento

Sistema desarrollado para centralizar la información del H. Ayuntamiento y mejorar el acceso a acuerdos y sesiones. Permite consultar documentos, organizar contenidos, administrar datos internos y mantener un registro claro de cada proceso. Su diseño está enfocado en transparencia, eficiencia y facilidad de uso para cualquier usuario autorizado.

Beaufort Ferreti

Sistema creado para centralizar la gestión de una tienda de ropa y optimizar la experiencia del cliente. Permite explorar colecciones, organizar catálogos, administrar inventarios y mantener un control claro de cada producto. Diseño enfocado en eficiencia, estética visual y facilidad de uso.

Recetario

Sistema creado para centralizar las recetas de mi mamá, quien es pastelera, y facilitar la organización de todos sus procesos. Permite consultar preparaciones, clasificar ingredientes, administrar notas personales y mantener un registro claro de cada creación.

Spectra

Sistema desarrollado para simplificar la creación de componentes dentro de proyectos web, permitiendo generar elementos visuales y funcionales de manera rápida y organizada. Facilita seleccionar estilos, estructuras y propiedades, ofreciendo un proceso más eficiente para desarrolladores.
