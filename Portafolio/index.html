Swoopy

Swoopy es una aplicación tipo red social donde los usuarios pueden interactuar mediante publicaciones, comentarios, reacciones y mensajes. Integra un sistema interno llamado Orbix que impulsa la participación dentro de la plataforma, además de un marketplace enfocado en la venta de productos entre usuarios. Todo está diseñado para ofrecer una experiencia rápida, intuitiva y fácil de usar.

Departamentos velamar

Este proyecto consiste en un sistema web que centraliza la administración y visualización de departamentos. El usuario puede explorar cada propiedad, revisar disponibilidad, consultar precios, ver imágenes y obtener información clave antes de tomar una decisión. Por otro lado, los administradores tienen acceso a un panel donde pueden modificar datos, actualizar inventarios, gestionar calendarios y mantener todo el contenido en sincronía. La arquitectura está desarrollada con herramientas modernas para lograr un rendimiento óptimo y una experiencia responsiva.

H. Ayuntamiento

Este sistema fue desarrollado para centralizar la información del H. Ayuntamiento y mejorar el acceso a los acuerdos y sesiones. Permite consultar documentos, organizar contenidos, administrar datos internos y mantener un registro claro de cada proceso. Su diseño está enfocado en la transparencia, la eficiencia y la facilidad de uso para cualquier usuario autorizado.

Beaufort Ferreti

Este sistema fue creado para centralizar la gestión de una tienda de ropa y optimizar la experiencia del cliente. Permite explorar colecciones, organizar catálogos, administrar inventarios y mantener un control claro de cada producto. Su diseño está enfocado en la eficiencia, la estética visual y la facilidad de uso para cualquier usuario que visite la plataforma.

Recetario

Este sistema fue creado para centralizar las recetas de mi mamá, quien es pastelera, y facilitar la organización de todos sus procesos. Permite consultar preparaciones, clasificar ingredientes, administrar notas personales y mantener un registro claro de cada creación. Su diseño está enfocado en la eficiencia, la practicidad y la facilidad de uso para que pueda gestionar sus recetas de manera rápida y ordenada.

Spectra

Este sistema fue desarrollado para simplificar la creación de componentes dentro de proyectos web, permitiendo generar elementos visuales y funcionales de manera rápida y organizada. Facilita seleccionar estilos, estructuras y propiedades sin necesidad de escribir cada parte manualmente, ofreciendo un proceso más eficiente para desarrolladores. Su enfoque está en la productividad, la claridad y la flexibilidad, brindando una herramienta que ayuda a construir interfaces completas de forma práctica, ordenada y adaptable a cualquier necesidad.
